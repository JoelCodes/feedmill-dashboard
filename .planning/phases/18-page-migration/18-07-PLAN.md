**If violations found:**
1. Note the file and line number

- Any color value (#hex) - must use --primary, --success, --error, etc.

- Border radius - must use --radius-X tokens

- Icon sizes: Create --icon-sm (14px), --icon-md (16px), --icon-lg (20px) tokens if missing
- Component-specific widths: Create --gauge-width (60px), --gauge-indicator (36px) if no token exists

**Goal: Zero raw violations. Any "exception" requires creating the missing token first.**
