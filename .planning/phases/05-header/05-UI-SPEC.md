## Interaction Patterns

### Global Search (Header)
- On input: Filter orders table on current page (if on /orders or /mill-production)
- On Enter: Navigate to /orders with search query applied
- Debounce: 300ms

### Notification Dropdown
- Open: Click bell icon
- Close: Click outside, click bell again, or click notification item
- Mark as read: Click notification item

### Settings Persistence
- All preferences persist to localStorage via useLocalStorage hook
- Changes apply on "Save Preferences" click
- Note: Theme/density visual changes are out of scope — preferences persist only
