    getBins() behavior:
    - Returns Promise<Bin[]>
    - Returns all bins from mockBins array
    - Each bin has fillPercentage and alertLevel

    getBinsByCustomerId(customerId: string) behavior:
    - Returns Promise<Bin[]>
    - Returns only bins where bin.customerId === customerId
    - Returns empty array if no bins exist for customer

    Test cases:
    - getBins() returns array with length > 0
    - Each bin has id property
    - Each bin has fillPercentage property (number)
    - Each bin has alertLevel property ("none" | "low" | "critical")
    - getBinsByCustomerId("CUST-001") returns only bins for that customer
    - getBinsByCustomerId("CUST-001") - all returned bins have customerId === "CUST-001"
    - getBinsByCustomerId("INVALID-CUSTOMER") returns empty array
    - Bin with fillPercentage < 20 has alertLevel "critical"
    - Bin with fillPercentage 20-39 has alertLevel "low"
    - Bin with fillPercentage >= 40 has alertLevel "none"
  </behavior>
