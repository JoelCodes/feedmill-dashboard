## Critical Gaps

### INT-01 / FLOW-01 — BLOCKER

**`src/components/ui/Timeline.tsx:123`** — `href` points to deleted route.

```js
href={`/orders?selected=${event.orderId}`}    // BROKEN: /orders is 404 after Phase 26
```

Should be:

```js
href={`/demo/orders?selected=${event.orderId}`}
```

A demo user on `/demo/customers/[id]` expanding an order activity event and clicking "View Order Details" navigates to a 404. This was missed by Phase 26's grep-based audit because Timeline.tsx is a UI primitive under `src/components/ui/` rather than under the demo route tree.

**Affected:** ROUTE-01 ("Existing pages moved to /demo/* subdirectory" — the page moved, but this consumer wasn't updated).

**Affected flow:** Demo customer → activity timeline → individual order detail.
