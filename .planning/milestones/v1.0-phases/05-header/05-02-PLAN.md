   - Add optional onSearch?: (query: string) => void

4. Add route-to-title mapping function (per D-10, RESEARCH.md):
   ```typescript
   const getPageTitle = (path: string): string => {
     if (path === '/') return 'Dashboard'
     if (path.startsWith('/orders')) return 'Orders'
     if (path.startsWith('/mill-production')) return 'Production'
     if (path.startsWith('/inventory')) return 'Inventory'
     if (path.startsWith('/shipments')) return 'Shipments'
     if (path.startsWith('/settings')) return 'Settings'
     return 'Dashboard'
   }
   ```

5. Component state:
   - const pathname = usePathname()
   - const router = useRouter()
   - const title = getPageTitle(pathname)
   - const [searchTerm, setSearchTerm] = useState('')
   - const debouncedSearchTerm = useDebounce(searchTerm, 300)
   - const [notifications, setNotifications] = useState<Notification[]>([])
   - const [isDropdownOpen, setIsDropdownOpen] = useState(false)
   - const [readNotificationIds, setReadNotificationIds] = useLocalStorage<string[]>('notifications-read', [])

6. Load notifications on mount:
   ```typescript
   useEffect(() => {
     getNotifications().then(setNotifications)
   }, [])
   ```

7. Search callback (debounced):
   ```typescript
   useEffect(() => {
     if (onSearch) {
       onSearch(debouncedSearchTerm)
     }
   }, [debouncedSearchTerm, onSearch])
   ```

8. Notification handlers:
   - Compute unreadCount with useMemo: notifications.filter(n => !readNotificationIds.includes(n.id)).length
   - handleMarkAsRead: add id to readNotificationIds if not present
   - handleClearAll: add all notification ids to readNotificationIds
   - toggleDropdown: setIsDropdownOpen(!isDropdownOpen)

9. Update JSX:
   - Title section uses computed title variable
   - Search input: value={searchTerm}, onChange={(e) => setSearchTerm(e.target.value)}
   - Settings button: onClick={() => router.push('/settings')} (per D-09)

   - Add badge to bell button per 05-UI-SPEC.md:

     - Show: only when unreadCount > 0
     - Display: unreadCount > 99 ? '99+' : unreadCount
   - Add NotificationDropdown below bell button:
     - notifications={notifications}

     - onClose={() => setIsDropdownOpen(false)}
