| Boundary | Description |
|----------|-------------|
| RSC (server) ↔ client component | `canEdit` is computed server-side via `await checkRole('mill_operator')` and passed as a serializable boolean prop; never recomputed in the browser |
| Page authentication gate | `await auth()` + `redirect('/sign-in')` on missing `userId` is the page's authentication check (server-side); middleware is the outer defense-in-depth (already in place per CONTEXT.md "Existing middleware") |
