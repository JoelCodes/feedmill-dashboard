## Spacing Scale

Declared values (all multiples of 4, established in Phase 16):

| Token | Value | Usage |
|-------|-------|-------|
| --space-1 | 4px | Icon gaps, inline padding |
| --space-2 | 8px | Compact element spacing |
| --space-3 | 12px | Medium-compact spacing |
| --space-4 | 16px | Default element spacing |
| --space-5 | 24px | Section padding |
| --space-6 | 32px | Layout gaps |
| --space-8 | 48px | Major section breaks |
| --space-10 | 64px | Page-level spacing |
| --space-12 | 96px | Extra large sections |

Exceptions: None. All spacing must use tokens from globals.css.

**Migration Note:** ESLint rule `no-hardcoded-values` blocks any px values. Use `bg-[var(--space-4)]` or Tailwind aliases (`--spacing-md`).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | --bg-page (#f8f9fa light, #1a202c dark) | Page backgrounds |
| Secondary (30%) | --bg-card (#ffffff light, #2d3748 dark) | Cards, panels, sidebar |
| Accent (10%) | --primary (#4fd1c5 light, #63b3ed dark) | Primary CTAs, filter pills (active), links, theme toggle active state |
| Success | --success (#48bb78 light, #68d391 dark) | StatusBadge (completed), BinGauge (>25%), success messages |
| Warning | --warning (#f59e0b light, #fbd38d dark) | StatusBadge (mixing), BinGauge (10-25%), warning alerts |
| Error | --error (#e53e3e light, #fc8181 dark) | StatusBadge (blocked), BinGauge (<10%), destructive actions, validation errors |

Accent reserved for: Primary action buttons, active filter pills, interactive state indicators (theme toggle), primary navigation active state.

**Source:** globals.css `:root` and `.dark` token definitions. Two-tier token architecture (primitives → semantic).

**Migration Note:** All hardcoded hex values must be replaced with token references. Use `bg-[var(--primary)]` syntax.
